Code Sample with Highlighting


def financial_future_value(P, r, n):
    # P = principal, r = rate, n = periods
    FV = P * (1 + r)**n
    return FV

print(financial_future_value(1000, 0.05, 10))

Financial Formula

Compound Interest:

\[ A = P(1 + \frac{r}{n})^{nt} \]

Where:

  • \( A \) = Final Amount
  • \( P \) = Principal
  • \( r \) = Interest Rate
  • \( n \) = Number of Compounds per Period
  • \( t \) = Time
